DOM is an interface for connecting html.The purpose of DOM is to get the elements easily by id,class etc.We can also add,create or remove any items.We can also style the elements by using DOM.

We can select HTML Elements by using:
getElementsByClassName
getElementById
querySelector
querySelectorAll etc

Event Bubble is an event where it starts work with child node and gradually it reaches parent node and work with it..

Callback function is a function that works inside of another function. We use it for such kind of work where we have to call the functions again again to get results and it executes after one task is over.
